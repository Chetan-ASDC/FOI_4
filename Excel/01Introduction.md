# Microsoft Excel – Basic Operations & Environment

---

## 1. Exploration of Excel Environment

### 1.1 Excel Workbook & Worksheet
- **Workbook:** An Excel file that contains one or more worksheets  
- **Worksheet:** A grid of rows and columns where data is entered  

### 1.2 Rows, Columns & Cells
- **Rows:** Horizontal (1, 2, 3, …)
- **Columns:** Vertical (A, B, C, …)
- **Cell:** Intersection of a row and a column (e.g., A1)

### 1.3 Excel Interface Components
- **Title Bar:** Displays file name
- **Ribbon:** Contains tabs and commands (Home, Insert, Page Layout, etc.)
- **Quick Access Toolbar:** Save, Undo, Redo
- **Formula Bar:** Shows and edits cell content
- **Status Bar:** Displays cell mode and quick calculations

---

## 2. Creating, Deleting & Manipulating a File

### 2.1 Creating a New Excel File
- Open Excel → **New** → Blank Workbook
- Shortcut: `Ctrl + N`

### 2.2 Saving a File
- **Save:** `Ctrl + S`
- **Save As:** Choose location, file name, and format (`.xlsx`)

### 2.3 Opening an Existing File
- **File → Open**
- Shortcut: `Ctrl + O`

### 2.4 Deleting an Excel File
- Delete from file location (Recycle Bin / Trash)
- Or right-click → Delete

### 2.5 Manipulating a File
- Rename workbook
- Move or copy file to another folder
- Change file format (`.xlsx`, `.csv`, `.pdf`)

### 3.3 Number Formatting

- Number

## 4. Insertion of Objects in Excel

### 4.1 Inserting Rows & Columns
- Right-click → Insert
- Home tab → Insert

### 4.2 Inserting Charts
- Bar chart
- Column chart
- Line chart
- Pie chart

### 4.3 Inserting Images & Shapes
- Insert → Pictures
- Insert → Shapes
- Insert → Icons

### 4.4 Inserting Tables
- Select data → Insert → Table
- Helps in sorting and filtering data

## 5. Summary

- Excel environment includes **workbooks, worksheets, ribbon, and formula bar**.
- Files can be **created, saved, renamed, and deleted** easily.
- Text and numbers can be formatted for better readability.
- Objects like **charts, images, and tables** enhance data presentation.
